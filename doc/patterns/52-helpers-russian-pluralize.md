# 52 — Хелперы: человекочитаемое время + русская плюрализация

## Что
View-хелперы для форматирования времени/длительности с правильными русскими словоформами (час/часа/часов).

## Реальный код
```ruby
module AnimeHelper
  def time_of_next_episode(anime)
    # ... вычисление статуса показа ...
    str = time_ago_in_words(episode_end, "%s назад")
    if str =~ /час/
      hours = str.match(/\d+/)[0].to_i
      # русская плюрализация: 1 час / 2 часа / 5 часов
      Russian.p(hours, "#{hours} час назад", "#{hours} часа назад", "#{hours} часов назад")
    else
      str
    end
  end

  def humanize_minutes(minutes)
    hours = (minutes / 60).floor
    text = hours > 0 ? "#{hours} #{t('anime_helper.hour', count: hours)}" : ''
    # ... + минуты
  end
end
```

## Как работает
- `Russian.p(count, form1, form2, form5)` — выбирает форму по русским правилам: 1→«час», 2-4→«часа», 5+→«часов» (гем `russian`, см. [38](38-i18n-russian.md)).
- `t('...', count:)` — стандартный i18n с плюрализацией для простых случаев.
- Форматирование «сейчас на ТВ / N часов назад / с HH:MM по HH:MM» — доменная логика отображения времени выхода серий.

## Зачем / где полезно
Презентационные мелочи, специфичные для языка: склонения числительных, человекочитаемые интервалы. В русском (и многих языках) плюрализация сложнее английского `s` — нужен явный выбор формы.

⚠️ Помечен `# TODO: remove this helper` — логику отображения лучше держать в декораторах/view-объектах ([39](39-decorator-caching.md), [43](43-view-objects.md)), а не в хелперах. Хороший пример «как НЕ надо» (логика в хелпере) рядом с «как надо».
