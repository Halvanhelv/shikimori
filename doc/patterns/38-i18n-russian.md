# 38 — Русский i18n + словоформы

## Что
Полноценная поддержка русского сверх перевода строк: падежи месяцев, склонения, транслит, двуязычные названия по предпочтению юзера.

## Реальный код
```ruby
# Gemfile — форк гема для русской морфологии
gem 'russian', github: 'morr/russian', branch: 'master'

# падежи месяцев (родительный для дат)
common_abbr_month_names: [nil, 'янв.', 'февр.', 'марта', 'апр.', 'мая', ...]

# выбор названия по локали + предпочтению юзера
def entry_name(entry, user)
  is_russian = I18n.russian? && (!user || user.preferences.russian_names?)
  is_russian && entry.russian.present? ? entry.russian : entry.name
end
```

## Как работает
- Гем `russian` даёт морфологию: падежи, склонения, правильные формы месяцев.
- `i18n-inflector` (форк) — сложная плюрализация для нескольких языков.
- `translit` — кириллица → латиница для имён файлов.
- Двуязычные сущности: показываем `russian` или `name` по локали и настройке юзера.

## Зачем / где полезно
Продакшн-поддержка не-английских языков с богатой морфологией. Учит: i18n ≠ только словарь строк — это падежи, склонения, словоформы, пользовательские предпочтения отображения.
