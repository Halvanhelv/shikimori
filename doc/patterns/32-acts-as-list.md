# 32 — `acts_as_list` (позиционирование)

## Что
Авто-управление порядком записей через колонку `position`, в рамках скоупа.

## Реальный код
```ruby
class Favourite < ApplicationRecord
  acts_as_list scope: %i[user_id linked_type kind]   # порядок внутри группы
  scope :ordered, -> { order :position }
end
```

## Как работает
- `scope:` — внутри какой группы считается позиция (тут: у юзера, по типу и виду избранного).
- Автоматом проставляет/сдвигает `position` при создании/удалении/перемещении.
- Методы: `move_higher`, `move_to_top`, `insert_at(n)` и т.п.

## Зачем / где полезно
Сортируемые списки: избранное, пункты меню, шаги, плейлисты, drag-and-drop порядок. Не надо руками пересчитывать позиции при вставке в середину. `scope` изолирует нумерацию по группам.
